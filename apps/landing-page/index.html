LDW Merge Skills API

Sistema completo de gerenciamento de Hosts, Games e Players

📌 Sobre o Projeto

Este projeto é uma API REST desenvolvida com Flask + SQLAlchemy, conectada a um banco MySQL. Ele permite o gerenciamento de Hosts (empresas), Games e Players através de rotas HTTP GET e POST.

⚙️ Tecnologias Utilizadas

  • ✔ Python 3.14
  • ✔ Flask
  • ✔ SQLAlchemy
  • ✔ MySQL
  • ✔ Flasgger (Swagger)
  • ✔ HTTP REST API
  • ✔ Flet (Frontend Desktop)
  • ✔ UV (gerenciador de dependências)

🔗 Rotas da API

GET /hosts

Lista todas as empresas host

POST /hosts

Cria uma nova empresa

GET /games

Lista todos os jogos

POST /games

Cria um novo jogo vinculado a um host

GET /players

Lista todos os jogadores

POST /players

Cria novos jogadores vinculados a um game

🚀 Como Executar o Projeto


# Backend
cd apps/backend
uv run flask --app src/app run

# Frontend (Flet)
cd apps/frontend
uv run python main.py

# Instalar dependências (UV)
uv sync

📦 Gerenciador de Dependências

Este projeto utiliza uv no lugar de pip para gerenciamento de dependências, proporcionando instalação mais rápida e eficiente.
